Prepare your environment

You can start using postgresql with docker. It's can best choice for development but for production needs better to use standalone installation.

docker run -p 5432:5432 -d postgres:12

# With expression

Also known as CTE (Common table expression), CTE allows you to write queries more clear using chain techies. Lets look an example above:

SELECT * FROM tablename
SELECT * FROM tablename

And with CTE we can write recursive expressions:

SELECT * FROM tablename

You can find details in documentation.
